Space browser

The Space browser allows you to

  1. View a table of entry types and the number of instances per type
  2. Iterate, view and delete Entry objects

Use the buttons below to query the JavaSpace.

  • Get entries/Refresh
    Queries the JavaSpace for a list of entries
  • Auto refresh
    Queries the JavaSpace every 5 seconds and updates the view

Viewing and deleting entries

Double click on a row to activate the Entry viewer, which allows you to step the through Entry objects for the selected type, see the field values and optionally delete the current Entry This option is NOT available when the auto refresh in running

Unusable entry count

If any entries returned by the JavaSpace throw an UnusableEntryException then they are added to a row called "unusable entry"

An UnusableEntryException occurs when the JavaSpace proxy cannot deserialize an Entry. Frequently the cause of this exception is due to an invalid codebase

WARNING: If the selected JavaSpace contains millions of entries, this is likely to cause an OutOfMemoryException in the browser
